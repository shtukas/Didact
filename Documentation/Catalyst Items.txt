Catalyst items are json object with at least a uuid ("uuid") and a miku type ("mikuType").

Cross types conventions (all optional):

    - field11             : coredata reference string
    - doNotShowUntil      : null or unixtime
    - tmpskip1            : "YYYY-MM-DD" skipped for the entire day
    - open-cycle-1143     : uuid of the open cycle that this item represents
    - donation-1605       : Array[String] uuids of donation target
    - ox1-0656            : Ox1
    - global-positioning  : Float or null # used for ordering and sorting
                            within the children of a core.
