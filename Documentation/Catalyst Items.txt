Catalyst items are json object with at least a uuid ("uuid") and a miku type ("mikuType").

Cross types conventions:

    - "field11"        : coredata reference string
    - "doNotShowUntil" : null or unixtime
    - "note"           : null of String
    - "tmpskip1"       : TmpSkip1

    - parent           : nil or Tx8
    - engine           : TxEngine
    - deadline         : TxDeadline (only NxTasks for the moment)
    - drivers          : null or Array[TxDriver]

Tx8 {uuid, position}

TxDriver: TxEngine | TxDeadline
