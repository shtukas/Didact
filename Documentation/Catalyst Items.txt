Catalyst items are json object with at least a uuid ("uuid") and a miku type ("mikuType").

Cross types conventions:

    - "field11"        : coredata reference string
    - "doNotShowUntil" : null or unixtime
    - "note"           : null of String
    - "tmpskip1"       : TmpSkip1

    - deadline         : nil or uuid of a NxDeadline
    - engine           : nil or uuid or a NxEngine
    - core             : nil or uuid of a NxCore
    - sequence         : nil or {uuid, position} 

Sequences are typically made of lines and tasks
