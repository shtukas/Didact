The Ax1Text is a type of attachement to a node. 
It contains (editable) text.
The toString function takes the first line.
It replaces regular attachment when we only wanted to store a line or some text.

{
    "uuid"        : String
    "clique"      : String
    "mikuType"    : "Ax1Text"
    "owneruuid"   : String
    "unixtime"    : Float
    "nhash"       : Nhash of the blobs of text
}
