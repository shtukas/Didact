
DxAntimatter: 
    "uuid"         : String
    "mikuType"     : "DxAntimatter"
    "unixtime"     : Float
    "datetime"     : DateTime Iso 8601 UTC Zulu
    "description"  : String
    "familyId"     : String
    "initialValue" : Float # in seconds

DxAntimatters are currently issued by Cores.

    - the familyId is a NxCore's uuid

    - The `value` is the `initialValue` + `the bank value` (the initial value is negatives)
When the value is positive we can merge two items of the same family
