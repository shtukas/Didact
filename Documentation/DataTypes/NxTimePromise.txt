
NxTimePromise: (Blade)
    "uuid"          : String
    "mikuType"      : "NxTimePromise"
    "unixtime"      : Float
    "datetime"      : Time.at(unixtime).utc.iso8601
    "targetuuid"    : String
    "value"         : Float

A NxTimePromise is a promise of a future bank operation against a particular item. When the time reaches `unixtime`, the value (algebraic value) is added to the items bank accounts.
