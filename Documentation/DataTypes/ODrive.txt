
An ODrive is one of the following objects in an open cycle's directory

ODrive: {
    "type" => "permanently-ignore"
}

ODrive: {
    "type"      => "weekly time target"
    "hours"     => Float
    "reseTtime" => Unixtime
}

ODrive: {
    "type"      => "coredata-target"
    "owneruuid" => string
}

They are stored in a .odrive file.
