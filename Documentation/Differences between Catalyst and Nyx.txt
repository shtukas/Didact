
Differences between Catalyst and Nyx

All the objects are essentialy on the same graph and there can be a link between any two, but in practice, Catalyst naturally manipulate some types dna Nyx manipulate the other types. The primary types are : 

Catalyst
    - Anniversary (special case independently stored)
    - TxDated
    - TxFloat
    - TxProject
    - TxTodo
    - Wave

Nyx
    - Nx100
    - NxTimeline
