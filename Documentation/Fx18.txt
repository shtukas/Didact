
# Fx18 Files

create table _fx18_ (
    _objectuuid_ text,
    _eventuuid_ text primary key,
    _eventTime_ float,
    _eventData2_ blob,
    _eventData3_ blob,
);

|-----------------------------|
| _eventData2_ | _eventData3_ |
|-----------------------------|
| <name>       | <value>      | ($3)
|-----------------------------|

($3) We JSON encode the value of an attribute.

So a Fx18 file is essentially just a log of events, setting attributes values
