
### PrimaryStructure.v1

{
    "mikuType"      : "PrimaryStructure.v1"
    "banking"       : nhash to a JSON encoded "PrimaryStructure.v1:Banking"
    "doNotShowUntil": nhash to a JSON encoded "PrimaryStructure.v1:DoNotShowUntil"
    "items"         : nhash to a JSON encoded "PrimaryStructure.v1:Items"
    "networkEdges"  : nhash to a JSON encoded "PrimaryStructure.v1:NetworkEdges"
}

LibrarianPointer {
    "mikuType" : "LibrarianPointer"
    "unixtime" : Float
    "nhash"    : nhash of the JSON encoding of a PrimaryStructure.v1
}

Given a LibrarianPointer and a PrimaryEvent, we get another LibrarianPointer.
