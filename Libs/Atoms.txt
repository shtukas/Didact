
### Atoms5

AtomType = "description-only" | "text" | "url" | "aion-point" | "unique-string" | "managed-folder" | "marble"

Atom {
    "uuid"        => String
    "unixtime"    => Float
    "type"        => AtomType
    "payload"     => String
}

Note: A marble Atom is a reference to a marble file. The atom's payload is the uuid of the marble file.

marble file:
    filename: nyx-marble.json
    content {
        "uuid" => String
    }
