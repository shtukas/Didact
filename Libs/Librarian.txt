
- Librarian files, aka Miku files, are put in Galaxy/Librarian/MikuFiles/<index>/

- Miku files are named <L22>.miku

- Miku files are sqlite3 files with two tables:
    create table _table1_ (_recorduuid_ text primary key, _unixtime_ float, _name_ string, _data_ blob) # attributes and other metadata
    create table _datablobs_ (_nhash_ string primary key, _data_ blob) # datablobs
    create table _notes_ (_uuid_ text primary key, _unixtime_ float, _text_ text)

- At _table1_ we expect the following attributes:
    "uuid"           : String
    "description"    : String
    "unixtime"       : Float # Often the creation time, sometimes used for ordering
    "datetime"       : DateTime Iso8601 # Contextual editable time
    "classification" : TxClassifier
    "atom"           : Atom

- The default function to create a new miku file is 
  Librarian::issueNewFile2(uuid, description, unixtime, datetime, classifier, atom)

  uuid-classifier-ordinal.sqlite3
