
- Librarian files, aka Miku files, are put in Galaxy/Librarian/Files/<index>/

- Miku files are named <L22>.miku

- Miku files are sqlite3 files with one table:
    create table _table1_ (_recorduuid_ text primary key, _unixtime_ float, _name_ string, _data_ blob)

MTxNote {
    uuid     : String
    unixtime : Unixtime
    text     : String
}

TxMiku {
    "uuid"           : String
    "filepath"       : String
    "version"        : "20220128"
    "description"    : String
    "unixtime"       : Float # Often the creation time, sometimes used for ordering
    "datetime"       : DateTime Iso8601 # Contextual editable time
    "classification" : TxClassifier
    "atom"           : Atom
    "notes"          : Array[MTxNote]
    "extras"         : Object # keys/values pairing for non standard attributes. 
}
